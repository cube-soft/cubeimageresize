■What's this
CubeImage Resize は画像のサイズ調整を行うためのソフトウェアです。

CubeImage Resize を使用するためには、Microsoft .NetFramework 2.0 がインストールされている必要があります。
Microsoft .NetFramework 2.0 は、以下の URL からダウンロードして下さい。
http://www.microsoft.com/downloads/details.aspx?FamilyId=0856eacb-4362-4b0d-8edd-aab15c5e04f5&displaylang=ja

■使用用法
まず、はじめに「サイズ設定」の「画面サイズ」から最終的にしたい画像のサイズを横×縦の順番に入力します。

次に出力される画像ファイルを画質を指定して出力するかファイルサイズを指定して出力するかを選択します。
画質を指定して出力する場合は0から100までの数字、もしくは高中低、もしくはその下のバーを動かして決めます。
ファイルサイズを指定する場合は何kB以内で出力するかを数字で入力します。

ここで、出力する画像に対して処理を施したい場合は画像効果から欲しい効果にチェックを入れます。

次に出力するファイル名をどうするか選択します。
新しくフォルダを作る場合は「新しいフォルダに保存」にチェックを入れフォルダ名を入力します。
ファイル名を変更する場合は「ファイル名にテキストを追加」にチェックを入れ追加する文字を入力、その文字をどこに追加するのをリストから選択します。

最後にサイズ調整を行いたい画像や画像を含むフォルダを「ここに画像データをドラッグ」のところにドラッグ＆ドロップすれば指定した画像が出力されます。

また、予めいくつかの画像サイズは登録されており「他のサイズより選ぶ」から選ぶことができます。
選んだサイズや各種設定は「設定を保存」を押すことで保存することができます。
